Project Description
A web-based smart transport management system that enables real-time bus tracking, delay handling, and structured communication between students and drivers through dual portals.

The Problem Statement

Students often face uncertainty about bus arrival times, while drivers lack clarity about delayed or absent students. There is no structured communication system, leading to confusion, unnecessary waiting, and inefficient attendance management.

The Solution

We developed a dual-portal web application (Driver Portal + Student Portal) that provides live trip monitoring, time-bound delay management (maximum 10 minutes), stop-wise student tracking, and automated delay expiry for better coordination and discipline.

View delayed students

Stop-wise present & absent count

Auto-expiry of delay after 10 minutes

Manual “Checked-In” confirmation

🎓 Student Portal

Login using college email

Select assigned bus

Live bus location (if tracking enabled)

ETA (Estimated Time of Arrival)

Stop status indicator (Not Reached / Approaching / Left)

Apply leave by selecting date

Apply delay (Maximum 10 minutes, no consecutive requests)

Delay expiry notification
